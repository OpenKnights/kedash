## ✨ Features

- 🪶 **轻量 & 模块化** – 支持按需导入与 Tree-shaking
- 🧠 **TypeScript 优先** – 完整类型提示，智能推导
- 🛠 **函数式设计** – 更简洁的编码体验
- ⚙️ **零依赖** – 无外部依赖，适用于浏览器和 Node.js
